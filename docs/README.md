# docs/ — Tài liệu bàn giao

Chứa **báo cáo Microsoft Word** và tài liệu bàn giao cho khách.

## Báo cáo Word cần có (theo yêu cầu khách)
1. Công nghệ sử dụng.
2. Kiến trúc ứng dụng.
3. Thiết kế cơ sở dữ liệu. → chưa có nháp, cần viết lại từ code thực tế (`../lib/data/`, `../lib/domain/`)
4. Vị trí lưu trữ dữ liệu.
5. Giải thích các chức năng chính. → xem [`csb-vocab-analysis/`](csb-vocab-analysis/)
6. Hướng dẫn sử dụng và triển khai.

> Viết bằng tiếng Việt. Nội dung tổng hợp lại từ mã nguồn thực tế (`../lib/`) và các tài liệu
> trong thư mục này (`csb-vocab-analysis/`, `artifact-design/`, `artifact-design-windows/`).
