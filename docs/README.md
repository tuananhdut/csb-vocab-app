# docs/ — Tài liệu bàn giao

Chứa **báo cáo Microsoft Word** và tài liệu bàn giao cho khách.

## Báo cáo Word cần có (theo yêu cầu khách)
1. Công nghệ sử dụng.
2. Kiến trúc ứng dụng.
3. Thiết kế cơ sở dữ liệu. → nháp: [`03-thiet-ke-co-so-du-lieu.md`](03-thiet-ke-co-so-du-lieu.md)
4. Vị trí lưu trữ dữ liệu. → có trong mục 5 của tài liệu trên
5. Giải thích các chức năng chính.
6. Hướng dẫn sử dụng và triển khai.

> Viết bằng tiếng Việt. Nội dung tổng hợp lại từ mã nguồn thực tế (`../lib/`) và các tài liệu
> thiết kế trong thư mục này (`03-thiet-ke-co-so-du-lieu.md`, `artifact-design/`, `artifact-design-windows/`).
